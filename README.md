# SYNOPSIS

Installs nginx from source using Sparrowdo

# Usage

    $ git clone https://github.com/melezhik/sparrowdo-nginx-source-install.git
    $ cd sparrowdo-nginx-source-install 
    $ sparrowdo --host=127.0.0.1

# Platforms supported

* debian
* ubuntu
* centOS
